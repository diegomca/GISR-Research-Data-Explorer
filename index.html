How to read the curves

Each line sorts nodes from lower to higher GISR centrality. Informative configurations produce monotonic curves with visible variability; nearly flat curves indicate weak discrimination because most nodes concentrate near the same centrality level.

Published data

27 synthetic networks, 2 diffusion models, 48 parameter combinations per network-model pair, and derived curve metrics for comparative analysis.
